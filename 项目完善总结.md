## 🛠️ 技术架构特点

### 前端技术栈
- **框架**: 微信小程序原生开发
- **UI组件**: 自定义组件库
- **状态管理**: 小程序原生数据管理
- **网络请求**: 封装的HTTP请求库

### 后端技术栈
- **运行环境**: Node.js 16+
- **Web框架**: Express.js
- **数据库**: MySQL 8.0 + Sequelize ORM
- **缓存**: Redis 6.0
- **身份认证**: JWT + Redis黑名单
- **文件上传**: Multer + 阿里云OSS

### 部署架构
- **容器化**: Docker + Docker Compose
- **反向代理**: Nginx
- **负载均衡**: 内置负载均衡
- **监控**: 健康检查和日志记录
