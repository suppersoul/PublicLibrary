## 技术特点

### 架构优势
1. **微服务架构**: 前后端分离，便于独立开发和部署
2. **容器化部署**: Docker容器化，支持快速部署和扩展
3. **缓存策略**: Redis缓存，提升系统性能
4. **安全防护**: 多层安全防护，保护用户数据
5. **日志监控**: 完整的日志系统，便于问题排查

### 技术栈
- **后端**: Node.js + Express.js + MySQL + Redis
- **前端**: 微信小程序原生开发
- **部署**: Docker + Docker Compose + Nginx
- **安全**: JWT + 限流 + CORS + Helmet

## 部署说明

### 开发环境
```bash
# 1. 配置环境变量
cd backend && cp .env.example .env

# 2. 安装依赖
npm install

# 3. 启动服务
npm run dev
```

### 生产环境
```bash
# 使用部署脚本
./scripts/deploy.sh prod deploy
```
